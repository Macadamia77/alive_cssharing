# 05. 브랜드 · CTA 레퍼런스

> **역할**: CTA에 쓸 실제 연락처·URL, "어떤 솔루션을 연결?" 판단용 솔루션 카탈로그의 단일 기준.
> 실제 값이 바뀌면 이 파일만 업데이트하면 전체 단계에 반영된다.

---

## 1. CTA 연락처

| 항목 | 값 |
|---|---|
| 대표전화 | **1522-5539** |
| 상담시간 | 월요일 09:00 ~ 18:00 (주말·공휴일 제외) |
| 기본 URL | **https://cssharing.com/** |

### 블로그 CTA URL 패턴

```
https://cssharing.com/?utm_source=naver_blog&utm_medium=organic&utm_campaign=[주제-슬러그]
```

- `[주제-슬러그]`는 각 주제를 영문/숫자 슬러그로 변환(예: AICC 관련 글 → `aicc`).
- 네이버 블로그 자체 추적기(bsa)는 UTM을 덮어씌우지 않는다.

## 2. 솔루션 카탈로그 (어떤 솔루션을 연결? 빠른 참고)

**이 표에 있는 명칭만 실존 서비스다.** 여기 없는 이름을 절대 만들어 쓰지 않는다 — 어미를 바꾸거나
("~솔루션"↔"~서비스") 두 단어를 조합해 새 이름을 지어내는 것도 금지. 애매하면 카테고리 표현으로
두루뭉술하게 쓰지 말고, 이 표에서 가장 가까운 실제 명칭 1개를 그대로 골라 쓴다.

| 카테고리 | 솔루션명 |
|---|---|
| CS 관련 솔루션 | CS 아웃소싱 / CS 쉐어링 솔루션 |
| 고객경험(CX) 솔루션 | 상담 아웃소싱 솔루션 / 품질 관리 솔루션 / VOC 관리 솔루션 / 고객감동 관리 솔루션 / 해피콜 운영 솔루션 / CX 솔루션플랫폼 |
| AI CS 솔루션 | OASIS AICC+IPCC / AI VOC / AI StandBy |

### 주제별 솔루션 연결 가이드

| 글 주제 키워드 | 적합 솔루션 |
|---|---|
| AICC, AI 전화봇, 챗봇 | **OASIS AICC+IPCC** |
| VOC, 품질 관리, 고객 만족도 개선 | AI VOC / VOC 관리 솔루션 |
| 상담사 채용, 서비스 위탁, 고객센터 아웃소싱 | CS 아웃소싱 / CS 쉐어링 솔루션 |
| 고객 응대, QA, 모니터링 | 고객감동 관리 솔루션 |
| 해피콜 운영 | 해피콜 운영 솔루션 / AI StandBy |

주제를 보고 가장 적합한 솔루션 1개를 정해 리서치 산출물에 기록한다. 위에 없는 새 주제가 나오면
카테고리를 참고해서 스스로 판단하고, 애매하면 "연결 필요"로 표시한다. 이 표 자체에 없는 서비스명이
필요하다고 판단되면 지어내지 말고 "연결 서비스 미확정 — 확인 필요"로 표시한다.

**개별 솔루션 상세 페이지 URL은 지금 없다.** 모든 솔루션은 기본 URL(`https://cssharing.com/`)로
링크하고, CTA 문구에 솔루션명을 기재한다.
